Ansible интеграция
Отправка уведомлений из плейбуков
Endpoint
POST http://notify-server:8080/api/ansible
X-API-Key: ansible-secret-key
Пример плейбука
- name: Уведомить пользователей
  uri:
    url: "http://notify:8080/api/ansible"
    method: POST
    headers:
      X-API-Key: "ansible-secret-key"
    body_format: json
    body:
      title:   "⚠️ Обновление"
      level:   "warning"
      type:    "popup"
      timeout: 15
      hosts:
        - "{{ inventory_hostname }}"
  become: false
